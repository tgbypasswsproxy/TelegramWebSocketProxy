# 🛡️ TgWsProxy: Обход блокировок Telegram в РФ

<img width="453" height="685" alt="image" src="https://github.com/user-attachments/assets/9dc81d54-7aad-4f5d-840c-458fbbac45fa" />

**TgWsProxy** — это легковесный локальный прокси-сервер, который решает проблемы с подключением к Telegram, возникающие у пользователей в Российской Федерации (РФ). Если официальный клиент Telegram работает медленно, не отправляет сообщения или не загружает медиа из-за действий систем DPI (Deep Packet Inspection) или IP-шейпинга со стороны провайдера — это средство поможет восстановить доступ.
